Cấu hình Firebase · Chỉ cần làm 1 lần
1
Truy cập console.firebase.google.com → Tạo project mới (miễn phí)
2
Vào Build → Realtime Database → Create database → Chọn Test mode
3
Vào Project Settings ⚙️ → Your apps → Web → Đăng ký app → Copy firebaseConfig và dán vào bên dưới
⚠️ Không tìm thấy databaseURL trong config.
Hãy đảm bảo bạn đã bật Realtime Database (không phải Firestore).
